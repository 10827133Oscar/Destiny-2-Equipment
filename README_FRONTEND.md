# Destiny 2 裝備系統 - Web 前端

這是一個暗色系主題的 Web 前端界面，用於管理 Destiny 2 裝備系統。

## 功能特點

- 🌙 **暗色系主題** - 護眼的深色界面設計
- 📱 **響應式設計** - 支持桌面和移動設備
- ⚡ **現代化 UI** - 流暢的動畫和交互體驗
- 🎯 **三大功能模塊**：
  - 添加裝備
  - 配置套裝
  - 查看倉庫

## 安裝與運行

### 1. 安裝依賴

```bash
pip install -r requirements.txt
```

### 2. 運行服務器

```bash
python app.py
```

### 3. 訪問前端

打開瀏覽器訪問：`http://localhost:5000`

## 使用說明

### 添加裝備

1. 選擇職業（泰坦/獵人/術士）
2. 選擇裝備類型（頭盔/臂鎧/胸鎧/護腿/職業物品）
3. 選擇裝備標籤（會自動顯示主詞條和副詞條信息）
4. 選擇隨機詞條（自動過濾掉主詞條和副詞條）
5. 可選：鎖定屬性（+5，會選擇懲罰屬性）
6. 可選：輸入套裝名稱
7. 點擊「添加裝備」按鈕

### 配置套裝

1. 選擇職業
2. 輸入目標屬性值（留空表示不要求該屬性）
3. 可選：使用異域裝備
   - 如果選擇使用異域裝備，需要配置：
     - 裝備類型
     - 裝備名稱
     - 屬性值（至少3個非零屬性）
     - 可選：裝備標籤
     - 裝備等級
4. 點擊「搜索裝備組合」按鈕
5. 系統會自動搜索最匹配的裝備組合

### 查看倉庫

1. 選擇職業（或選擇「全部職業」查看所有）
2. 點擊「刷新」按鈕
3. 查看裝備列表，包括：
   - 裝備名稱和類型
   - 裝備 ID
   - 標籤、等級、鎖定/懲罰屬性
   - 套裝名稱
   - 所有屬性值

## 技術棧

- **後端**: Flask (Python)
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **樣式**: 自定義暗色系主題

## 顏色主題

- 背景色：深灰藍色系 (#0d1117, #161b22, #21262d)
- 文字色：淺灰色系 (#c9d1d9, #8b949e)
- 強調色：藍色系 (#58a6ff, #1f6feb)
- 成功/錯誤：綠色/紅色 (#3fb950, #f85149)

## 注意事項

- 確保所有後端 Python 文件（equipment.py, equipment_manager.py 等）都在同一目錄
- 服務器運行在 `http://localhost:5000`，如需修改請編輯 `app.py`
- 前端使用現代瀏覽器 API，建議使用 Chrome、Firefox、Edge 等現代瀏覽器
