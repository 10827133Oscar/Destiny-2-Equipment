# Destiny 2 裝備系統與倉庫管理

這是一個專為 Destiny 2 設計的裝備管理與數值計算系統，可以幫助玩家快速計算裝備組合的屬性數值。

## 功能概述

- **裝備管理系統**：管理各種裝備（頭盔、臂鎧、胸鎧、護腿、職業物品）
- **倉庫系統**：儲存和管理所有裝備，支援三個職業（泰坦、獵人、術士）
- **數值計算器**：快速計算裝備組合的總屬性
- **套裝效果計算**：自動計算套裝加成效果
- **組合推薦**：根據目標屬性推薦最佳裝備組合

## 系統架構

- `equipment.py` - 裝備類別定義
- `inventory.py` - 倉庫管理系統
- `calculator.py` - 數值計算器
- `equipment_manager.py` - 裝備管理器（簡化操作接口）
- `classes.py` - 職業定義
- `main.py` - 主程式

## 安裝與使用

```bash
# 運行主程式
python main.py
```

## 功能說明

### 裝備屬性
系統支援以下 6 種屬性：
- 生命值
- 近戰
- 手榴彈
- 超能力
- 職業
- 武器

### 裝備標籤
每個裝備都有一個標籤，決定了主詞條和副詞條：
- **堡壘**：主詞條(生命值)，副詞條(職業)
- **赤拳互鬥**：主詞條(近戰)，副詞條(生命值)
- **榴彈兵**：主詞條(手榴彈)，副詞條(超能力)
- **至高典範**：主詞條(超能力)，副詞條(近戰)
- **戰術家**：主詞條(職業)，副詞條(武器)
- **槍手**：主詞條(武器)，副詞條(手榴彈)

### 詞條類型
每個裝備有 6 個屬性，分別對應不同的詞條類型：
- **主詞條**：基礎值 30（對應標籤的主詞條）
- **副詞條**：基礎值 25（對應標籤的副詞條）
- **隨機詞條**：基礎值 20（玩家選擇的隨機詞條）
- **補充詞條**：基礎值 0，升級後每級 +1（最多 +5）

### 職業系統
系統支援三個職業：
- 泰坦
- 獵人
- 術士

每個裝備可以限制特定職業使用，也可以設置為通用裝備（所有職業可用）。

### 裝備類型
- 頭盔
- 臂鎧
- 胸鎧
- 護腿
- 職業物品

### 鎖定系統
裝備可以鎖定特定屬性（+5），同時選擇一個懲罰屬性（-5）。這在配置套裝時可以自動優化。

### 套裝系統
裝備可以屬於特定套裝，系統會自動計算套裝加成效果。
