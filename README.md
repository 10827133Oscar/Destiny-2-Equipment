# Destiny 2 裝備系統與倉庫管理

這是一個專為 Destiny 2 設計的裝備管理與數值計算系統，可以幫助玩家快速計算裝備組合的屬性數值。

## 功能概述

- **裝備管理系統**：管理武器、護甲等各種裝備
- **倉庫系統**：儲存和管理所有裝備
- **數值計算器**：快速計算裝備組合的總屬性
- **套裝效果計算**：自動計算套裝加成效果
- **組合推薦**：根據需求推薦最佳裝備組合

## 系統架構

- `equipment.py` - 裝備類別定義
- `inventory.py` - 倉庫管理系統
- `calculator.py` - 數值計算器
- `main.py` - 主程式與範例

## 安裝與使用

```bash
# 運行範例程式
python main.py
```

## 功能說明

### 裝備屬性
支援 Destiny 2 的各種屬性：
- 移動 (Mobility)
- 韌性 (Resilience)
- 恢復 (Recovery)
- 紀律 (Discipline)
- 智力 (Intellect)
- 力量 (Strength)

### 套裝系統
支援多層級套裝效果，可設定不同件數的加成效果。
